You’re looking at Natalie Thurlby’s thesis as a jupyter book.

What is a Jupyter Book?¶

This jupyter book thesis is made up of markdown files and jupyter notebooks, which are kept in this GitHub repo. The code for all graphs in this thesis are available there (as jupyter notebooks), which you can run for yourself using binder if you like. This jupyter book thesis is therefore as up-to-date and as reproducible as possible.

Contents and navigation¶

You can use the sidebar on the left to navigate to a chapter or section, while the sidebar on the right navigates more finely within a “page” of the book. If you’d like an overview of the whole book, you can look at the table of contents.

At the bottom of each page are buttons which let you go to the previous or next sections.
